## ‚úÖ Week 1: Setup (Day 1-2)

### Getting Started
- [ ] Download TillSave app (iOS/Android)
- [ ] Sign up with your phone number
- [ ] Verify OTP (check SMS)
- [ ] Create your 4-digit PIN (write it down safely!)
- [ ] Enter your name

### Create Your First Group
- [ ] Click "Create New Group"
- [ ] Enter group name (e.g., "Monday Savings Circle")
- [ ] Get your 6-digit group code (e.g., A1B2C3)
- [ ] **IMPORTANT**: Write down this code! You'll need it
- [ ] Take screenshot of code
- [ ] Share via WhatsApp, SMS, or in person with members

### Add Members
- [ ] Share your group code with members
- [ ] Members join using Menu ‚Üí "Join Group" ‚Üí Enter code
- [ ] Check Members tab to see who joined
- [ ] Wait for at least 2 members before starting
- [ ] Minimum: 3 people (you + 2 members) recommended

---

## ‚úÖ Week 1-2: Daily Operations (Day 3-14)

### Recording Payments Daily
**When member gives you money, immediately do**:

1. [ ] Open TillSave app
2. [ ] Select your group
3. [ ] Click "Record Payment"
4. [ ] **Member Name**: Select from list
5. [ ] **Amount**: Type exact amount received
6. [ ] **Currency**: Choose (RWF, USD, EUR, etc.)
7. [ ] **Date**: Confirm it's today (or correct if different)
8. [ ] **Note**: Optional (e.g., "Cash from John")
9. [ ] Click "Record Payment"
10. [ ] ‚úÖ Confirm it shows in the list

**Daily Habit**: Record at same time each day (morning or evening)

### Check Dashboard Daily
- [ ] View total saved so far
- [ ] Check member contributions
- [ ] Note any members who haven't contributed yet
- [ ] Follow up with those members (optional)

---

## ‚úÖ Week 3-4: End of Cycle (Day 21-30)

### Before Finalizing (Important!)

1. [ ] **Verify Total Money**: Count physical cash you have
2. [ ] **Check Dashboard**: Compare to TillSave total
3. [ ] **Investigate Gaps**: If numbers don't match, find out why
4. [ ] **All Payments Recorded**: Make sure every payment is in the system

### Finalize Payout Process

**Step 1**: Open Group ‚Üí Click "End Cycle & Payout"

**Step 2**: Review Payout Breakdown
- [ ] Check total saved amount
- [ ] Check your commission (1 day per member)
- [ ] Check each member's payout amount
- [ ] Verify calculations look correct

**Step 3**: Click "Finalize" Button
- [ ] **Confirmation Modal Opens** (NEW!)
- [ ] Review one more time
- [ ] Check member names and amounts
- [ ] If something is wrong: Click "Cancel" ‚Üí Fix it
- [ ] If everything is correct: Click "Confirm Finalize"

**Step 4**: üéâ Celebration Screen Appears
- Shows cycle completed
- Two buttons appear:
  - [ ] **"View Payout Report"** ‚Üí Preview PDF
  - [ ] **"Start Cycle 2"** ‚Üí Begin next cycle

### Print/Save Report

**Before Starting Next Cycle**:
1. [ ] Click "View Payout Report"
2. [ ] Review the report
3. [ ] Print: Ctrl+P (Windows) or Cmd+P (Mac)
4. [ ] Save as PDF with name: "TillSave_[GroupName]_Cycle1_Dec2025.pdf"
5. [ ] Keep for records

### Distribute Payouts to Members

**Method Options**:
- [ ] Cash in hand (if you manage it)
- [ ] Bank transfer
- [ ] Mobile money
- [ ] Next meeting

**Best Practice**:
1. [ ] Have report printed
2. [ ] Show each member their amount
3. [ ] Get signature on report if paying in cash
4. [ ] Confirm they received correct amount
5. [ ] Keep signed report for records

## ‚úÖ After Finalization: Start Next Cycle

### Start Cycle 2

From celebration screen:
1. [ ] Click "Start Cycle 2"
2. [ ] Confirm reset (cycle counter increments)
3. [ ] Dates automatically set to today
4. [ ] ‚úÖ Cycle 2 is live!

## üÜò Troubleshooting

### "Member can't join group"
- [ ] Confirm they have 6-digit code
- [ ] Check spelling (case-insensitive)
- [ ] They need to: Menu ‚Üí Join Group ‚Üí Enter code
- [ ] Have them try again
- [ ] ‚Üí See [03-ORGANIZER_FAQ.md](03-ORGANIZER_FAQ.md) Q2

### "Payment recorded but member says they didn't pay"
- [ ] Ask them when they think they paid
- [ ] Check if date is correct
- [ ] Ask if it's the right amount
- [ ] ‚Üí See [03-ORGANIZER_FAQ.md](03-ORGANIZER_FAQ.md) Q5

### "App says 'syncing' forever"
- [ ] Check internet connection
- [ ] Wait 5 minutes
- [ ] Restart app if needed
- [ ] ‚Üí See [03-ORGANIZER_FAQ.md](03-ORGANIZER_FAQ.md) Q18

### "I forgot my PIN"
- [ ] Click "Forgot PIN" on login
- [ ] Verify phone number with OTP
- [ ] Create new PIN

### "Can't see a member's payment history"
- [ ] Go to group
- [ ] Click on member name
- [ ] Their payment history shows

## üí° Pro Tips

### Best Practices
1. **Record Daily**: Don't wait until end of cycle
2. **Backup**: Keep digital + printed records
3. **Communicate**: Tell members the rules upfront
4. **Consistent**: Same day, same process every cycle
5. **Transparent**: Show dashboard to members sometimes

### Common Mistakes to Avoid
- ‚ùå Forgetting to record a payment
- ‚ùå Confusing currencies
- ‚ùå Not telling members when to pay
- ‚ùå Losing the printed report
- ‚ùå Not starting next cycle immediately

### How to Explain to Members
```
"TillSave is a digital savings group. 
We meet every 30 days, you give me money, 
the app keeps track, and at the end of the cycle, 
everyone gets their savings back (minus my 1-day fee for managing).

Easy, transparent, secure."
```
